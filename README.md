# KBO-stats-To-Game
2022-2 WebPython Programming Term project

### 개요
* 2021년 KBO 타자들의 기록을 통해 게임 능력치를 설정하여 게임 "마구마구"의 능력치를 비교해 보기

### 동작 내용
* 2021년 KBO 타자의 기록을 분석하여 파워, 컨텍트, 스피드, 스로잉, 수비 능력치를 설정
* 게임 내 능력치와 비교
